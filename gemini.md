# Gemini向け 連想語補強指示

## 目的

絵文字を「読み」ではなく連想で検索できるように、`assoc_ja` を広げてください。

重要: 最初から `tags_ja`、`scenes_ja`、`tone_ja` に分類しなくてよいです。まずは連想語の大きなプールである `assoc_ja` を増やします。分類はPythonスクリプトで後から行います。

## 入力形式

入力はJSONLです。1行が1絵文字です。

```jsonl
{"emoji":"🦁","codepoints":"1F981","name_en":"lion","name_ja":"ライオンの顔","category_ja":"動物と自然","subcategory":"animal-mammal","subcategory_ja":"哺乳類","assoc_ja":["動物","ライオン"],"tags_ja":["動物","ライオン"],"scenes_ja":["動物園"],"tone_ja":["力強い"]}
```

## 出力形式

入力と同じ順番で、JSONLのパッチだけを返してください。CSV全体は返さないでください。

```jsonl
{"emoji":"🦁","name_ja":"ライオンの顔","assoc_ja_add":["百獣の王","サファリ","たてがみ","獅子座","星座","占い","動物占い","性格診断","堂々","力強い","勇ましい","王者","ワイルド"],"assoc_ja_remove":[]}
```

使えるキー:

- `emoji`: 必須
- `name_ja`: 変更したい場合だけ
- `assoc_ja_add`: 追加する連想語
- `assoc_ja_remove`: 削除する連想語

## やること

各絵文字について、検索者が思いつきそうな言葉を広げてください。

含めてよいもの:

- 物体名、生き物名、感情、季節、色、用途、比喩
- 日本語の言い換え
- 投稿やチャットで使う場面
- 絵文字を使ったときの雰囲気
- 文化的な使われ方
- 占い、星座、干支、動物占い、スピリチュアルの関連語

避けるもの:

- どの絵文字にも付く薄い語の増やしすぎ
- 意味が遠すぎる語
- 既存語の単純な言い換えだけで埋めること
- 後半ほど手を抜いて同じ語を繰り返すこと

目安:

- `assoc_ja_add` は通常5から15個
- 多義的な絵文字、占い・季節・文化に関わる絵文字は20個程度まで可
- `tags_ja`、`scenes_ja`、`tone_ja` の分類は不要

## name_ja

日本語名が英語のまま、または長すぎる場合だけ直してください。

方針:

- 日本語で自然に短く
- 直訳より、検索結果で見て理解しやすい名前
- 絵文字の用途が分かる名前

例:

- `grinning face with sweat` -> `汗笑い`
- `face with tears of joy` -> `嬉し泣き笑い`
- `lion` -> `ライオンの顔`
